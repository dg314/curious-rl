# Curious RL

## Setup

Create environment:
- `conda env create -f environment.yml`
- `conda activate Curious_RL`

Update environment:
- `conda env remove --name Curious_RL`
- `conda env create -f environment.yml`
- `conda activate Curious_RL`

Deactivate environment:
- `conda deactivate`

FFmpeg:
- `brew install ffmpeg`
